Wie kann ich die App deinstallieren?
Gehe zu Einstellungen > Sicherheit > Geräteadministratoren und AcDisplay deaktivieren. Jetzt können Sie AcDisplay deinstallieren.

How to dismiss the permanent notifications of active mode or the lockscreen?
Die Benachrichtigung ist notwendig damit AcDisplay nicht vom System beendet wird. Du kannst sie aber entfernen (Dann wird die Test-Benachrichtigung nicht mehr funktionieren!), dazu öffne die App-Info und deaktiviere die Benachrichtigungen der App.

Ich habe einen Fehler in der Übersetzung gefunden!
Danke, Sie können uns dabei ganz einfach helfen. Öffne Sie bitte die App-Seite auf crowdin und schlagen Sie eine bessere Übersetzung vor.

Wie verwende ich die Sperrliste?
Um die Sperrliste verwenden zu können müssen Sie diese aktivieren und dann konfigurieren. Um sie zu aktivieren, bitte einfach auf den Schaltknopf klicken. Um sie zu konfigurieren, bitte auf das Element klicke das Sie bearbeiten wollen.
