How can I uninstall this app?
Go to Settings > Security > Device Administrators and disable AcDisplay. You can now uninstall AcDisplay normally.

How to dismiss the permanent notifications of active mode or the lockscreen?
The notification is necessary to ensure AcDisplay isn't killed by the system. However, to hide it safely (sending test notification won't function anymore!) launch app info and disable showing notifications.

When the secure lockscreen (pattern or PIN) will be implemented?
This is not possible to create a custom secure lockscreen for Android. The other apps that have this feature are only pretending to be secure, but in real world there's always a way to bypass them.
